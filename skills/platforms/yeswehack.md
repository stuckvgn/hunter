# YesWeHack — platform rules

Smallest of the four platforms — 70 programs, 67 paying. European, less saturated.

## Researcher identity

Need a YesWeHack account (`yeswehack.com`). Sam's identity handles this.

## Scope page structure

URLs are `https://yeswehack.com/programs/<id>`. Sections:
- **Program scope** — in/out scope with types
- **Rules** — policy, automation rules, payout tiers
- **Activities** — recent program updates, bonuses

## Automation stance

Variable — YesWeHack hosts a lot of government / healthcare / defense programs with strict rules. Always read the Rules section carefully. Some French-language programs have their rules in French; translate before acting.

## Submission format

Web-form based. Markdown-friendly.

## Platform quirks

- YesWeHack-exclusive programs (rare) — these attract fewer hunters because of the platform's smaller user base.
- Runs "Dojo" CTF-style engagements sometimes — can be practice ground.
- Some government programs have quote-gaged legal frameworks; skip if unsure.
